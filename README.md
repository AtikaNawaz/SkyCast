# ✨ Overview

SkyCast is a modern weather application built with **HTML**, **CSS**, and **JavaScript**, powered by **WeatherAPI**.

It provides real-time weather information with a polished interface, animated weather effects, dynamic themes, and a fully responsive design to deliver an engaging user experience.

- Recent Search History
